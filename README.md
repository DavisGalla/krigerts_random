# Funkcionālās iezīmes

Mazs Vite + React projekts ar tulkojumu tabulu (“Gherkin termini”).

## Palaišana lokāli

```bash
npm install
npm run dev
```

## Būvēšana

```bash
npm run build
npm run preview
```

## Izvēršana (Deploy)

### Vercel

- **Framework Preset**: Vite
- **Build Command**: `npm run build`
- **Output Directory**: `dist`

### Cloudflare Pages

- **Framework preset**: Vite
- **Build command**: `npm run build`
- **Build output directory**: `dist`
